# get_specs
Get your computers specs

# Installation
### Linux
```
$pip install bs4
$pip install psutil
$pip install cpuinfo
$pip install GPUtil
$pip install platform
```
### Windows
```
>pip install bs4
>pip install psutil
>pip install cpuinfo
>pip install GPUtil
>pip install platform
```

### Simply run the program and it will output your specs
